Latest version of ADMIXTOOLS is up!

[![Screenshot 2016-06-13 18.15.37](https://i0.wp.com/www.unz.com/wp-content/uploads/2016/06/Screenshot-2016-06-13-18.15.37.png?resize=620%2C130)![Screenshot 2016-06-13 18.15.37](https://i0.wp.com/www.unz.com/wp-content/uploads/2016/06/Screenshot-2016-06-13-18.15.37.png?resize=620%2C130)](https://github.com/DReichLab/AdmixTools)

Just been notified that there’s an update to [Admixtools](https://github.com/DReichLab/AdmixTools) GitHub. Also, of interest to some: “This contains a public domain version of qpGraph, using the GNU subroutine library (gsl).” You can find [example data](https://genetics.med.harvard.edu/reich/Reich_Lab/Software_files/AdmixTools_Example_Data.tar.gz) the Reich lab website.

(related note, also might check out Alicia Martin’s [GitHub](https://github.com/armartin/ancestry_pipeline), it has a lot of local ancestry stuff)
