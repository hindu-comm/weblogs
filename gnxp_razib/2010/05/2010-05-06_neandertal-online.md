Neandertal online

Here’s the [UCSC Genome Browser page for the Neandertal genome](http://genome.ucsc.edu/Neandertal/). [Ensembl](http://uswest.ensembl.org/index.html) is supposed to have something up soon, but I don’t see it. Here’s the [Anfo Short Read Aligner/Mapper](http://bioinf.eva.mpg.de/anfo/); you can download it on that page. Page 21 of the online supplement [has some configuration file code](http://www.sciencemag.org/cgi/data/328/5979/710/DC1/1) on it which might be useful. Also, make sure to check out the [supplements](http://www.sciencemag.org/cgi/content/full/328/5979/710/DC1).
