Celestial Dialogues: PañcāṅgaSadas, Kali 5125

Every year since Kali 4979 (Śaka 1799/AD 1877), the KāñcīKāmakoṭiPīṭha conducts a PañcāṅgaSadas (meeting of astronomers and theologists) to determine the dates of ritual observances based on the Vedic luni-solar almanac (pañcāṅga). [This year’s (Kali 5125/Śaka 1945/AD 2023) sadas](https://www.kamakoti.org/kamakoti/news/2023/annual-panchnga-58532-20230808.html), held in Vārāṇasī, was the first to have participants from all over the subcontinent. This is a versified summary of some of the discussions in the sadas related to the mathematics of planetary computations, to the extent I understood them.

## *॥परिचयः॥*

``` wp-block-verse काशन्ते प्रभया यस्य सूर्यचन्द्रादयो दिवि। वन्देऽहं विश्वनाथं तं त्रैलोक्यस्यादिकारणम्॥ भास्करादीन्ग्रहान्नत्वा सङ्क्षेपेण प्रलिख्यते। ज्योतिर्गणितचर्चानां सारञ्ज्यौतिषमेलने॥

ऋषिभिर्या स्तुता भूमिः पुराणैर्या च शंसिता। उत्पत्तिर्विस्मृता यस्याः कालचक्रस्य वर्तनात्॥ तपोभूमिः पवित्राणां पवित्री पापकर्मणाम्। पुण्यदा तीर्थयातॄणां विद्यादा ब्रह्मचारिणाम्॥ पूता साक्षाद्धि जाह्नव्या स्मृता या मोक्षदा भुवि। तामहं विश्वनाथस्य पवित्रां नगरीं ययौ॥ नवम्यां श्रावणे कृष्णे बाणाब्धिग्रहभूशके। आयातः काशिकादेशे ग्रस्ताब्दसुरवर्त्मना॥ पञ्चाङ्गविषये ह्यत्र काञ्च्याचार्यप्रयोजिता। सद आभारताद्यत्र ज्योतिषज्ञास्समागताः॥ दशम्यामुदयादूर्ध्वं गतासु नवनाडिषु। चर्चारभत विद्वत्सु पर्वणां दिननिर्णये॥ नानाग्रन्थान्समन्वीक्ष्य सन्दिग्धानां विमर्शने। विद्वांसौ वैदिकाचारौ धर्मशास्त्रविचक्षणौ॥ द्रविडवाजपेय्यार्यौ धियाब्रूतां हि निर्णयान्। यत्र नासीत्प्रमाणं स्वं स्वीकृतं वाक्यमन्यतः॥ सरस्वत्यर्चनाप्रश्ने केरलेभ्यो यथोद्धृतम्॥ ```

## ***॥प्राचीनगणिताः॥***

``` wp-block-verse अन्या अन्येषु देशेषु प्राचीनगणना युताः। कर्णाटैः पुष्पवन्तौ तु ग्रहलाघवसाधितौ॥ दृश्येनार्यभटीयेन वेतरग्रहसाधनम्। सूर्यसिद्धान्ततस्सिद्धाश्शृङ्गेर्यां तिथयस्तथा॥ द्रविडानां प्रसिद्धौ द्वौ मठसर्पाभिधौ तथा। श्रीरङ्गेषु तथान्यच्च चिरकालात्प्रकाशितम्॥ यथादिष्टमधीशेन मठाख्यं दृश्यतः कृतम्। वररुच्युदितैर्वाक्यैस्सर्पश्रीरङ्गसाधनम्॥ काशिकासु प्रयुज्येते पद्धती ग्रहसाधने। मकरन्दीयसारण्यास्सूर्यसिद्धान्ततस्तथा॥ पुरा परहिताख्यं यद्धरिदत्तेन संस्कृतम्। तस्य संशोधनं यच्च दृगाख्यं पारमेश्वरम्॥ तस्यापि शोधनं कृत्वा युज्यते केरलैरिह। भगणा नूतनाः क्षिप्ता आङ्ग्लमुक्तदिनं ध्रुवः॥ प्रोक्तानि नववाक्यानि तथा कटपयादिभिः। एवं स्फुटग्रहास्सिद्धा रीत्या प्राचीनया बुधैः॥ आधिक्येनान्यदेशेषु दृश्यमेव प्रयुज्यते॥ ```

## ***॥देशान्तरसंस्कारः॥***

``` wp-block-verse देशान्तरे तु संस्कारे तदारेभे विमर्शनम्। करणाल्लभ्यमाना या पूर्वरीतिषु कुत्र ताः। अवन्त्यां निजदेशे वा घटिका उदयादयः॥ आभारतेष्ववन्त्यां वै स्वीकृता गणकैः समैः। द्रविडेषु त्वहो नैवं स्वदेशीया इति स्मृताः॥ देशान्तरादयस्तैस्तु करणे हि कृता मताः। सदस्यो विषयेऽमुष्मिन्नपृच्छत्सदसस्पतिम्।। देशान्तराय दत्ता हि पञ्चाङ्गेषु तु सारणी। यया साध्या अभीष्टस्य देशस्य घटिकादयः॥ किञ्जनैरेव नो शक्तं कर्तुं संस्कारमीदृशम्। एवं पृष्टस्सदस्येन प्रत्युवाच सभापतिः॥ तद्दुर्गणनसंस्कारं सामान्यैर्नैव शक्यते। पञ्चाङ्गकृद्भिरेवातस्संस्कृत्य काश्यतां स्फुटम्॥ ```

## ***॥सायननिरयणविमर्शनम्॥***

``` wp-block-verse अपराह्णेऽभवच्चर्चा सायनस्य प्रयोजने। तत्र धीमान्यथाशास्त्रं रमणाचार्य अब्रवीत्॥ लोके निरयणान्येव पर्वाण्यद्य यशस्विनः। सायनानि च पर्वाणि पूर्वैराचरितानि वै॥ कालेन तानि लुप्तान्याचरितव्यान्यतः पुनः॥ ```

### *॥**निरयणव्याप्तिः**॥*

``` wp-block-verse केचिद्वदन्ति सङ्ख्यानं केवलं सायनं भवेत्। वेदे यतो हि विद्यन्ते शब्दा मेषादयो न वै॥ तदप्यसाधु मन्येत यद्यप्येवं सति श्रुतौ। नक्षत्रपूर्णिमे स्तो ये निरयणस्य लक्षणे॥ ```

### *॥**चान्द्रमासानां निरयणानेतव्यत्वम्**॥*

``` wp-block-verse युक्ताश्चित्रादिभिर्ह्येव पौर्णमास्या विदश्श्रुतौ। राका च यत्र चित्रायां सो मासश्चैत्र उच्यते॥ यत्सास्मिन्पौर्णमासीति सञ्ज्ञायामिति पाणिनेः। यद्यपि सर्वदा न स्याच्चित्रायाञ्चैत्रपूर्णिमा। नातिदूरं भ्रमेत्तस्या ग्राह्यस्तद्धीतिकारतः॥ एवं निरयणेनैव गणितेन प्रसाध्यते। त्यजेद्राका यतो भानि सायने गणने कृते॥ ```

### ***॥राशीनां निरयणानेतव्यत्वम्॥***

``` wp-block-verse मेषप्रभृतयश्चैव ये शास्त्रेषु प्रकीर्तिताः। नक्षत्राणां समूहो हि राशिरित्यभिधस्स्मृतः॥ भानि मेषाकृतौ यत्र तत्र मेषो नभेऽङ्कितः। मेषाद्यास्सायना ये स्युः केवलं व्यावहारिकाः॥

केवलसायने यान्ति राशिचान्द्रा अनर्थताम्। तेऽतो निरयणा ग्राह्या नातो निरयणं त्यजेत्॥ ```

### ***॥सायनव्याप्तिः॥***

``` wp-block-verse परन्त्वानयने योज्या केषाञ्चित्सायना प्रथा। यथा पीयूषधारायाः कर्त्रा श्लोकेन काशितम्॥ अयनांशाः प्रदातव्या लग्ने क्रान्तौ चरागमे। सत्रिभे वित्रिभे पाते तथा दृक्कर्मपातयोः॥ ```

### ***॥ऋतूनां सायनानेतव्यत्वम्॥***

``` wp-block-verse श्रुताश्च वेदवाक्येषु षड्द्विमासमितर्तवः। ऋतुभ्योऽभिहिता वेदे मासा मध्वादयस्तथा॥ नामान्यनुगुणान्येषां माधुर्यं मधुलक्षणम्। एते यद्यपि चान्द्रा वै ये स्युर्निरयणाः पुनः॥ ऋतूनां भ्रमणं किन्तु सूर्यायनानुरागि वै। अतस्स्याद्गणनं नूनमृतूनां सायनं सदा॥ मधुमाधवमासौ च चैत्रवैशाखिकाविति। विषुवन्तं तु मेषस्थं मत्वा पूर्वैः प्रकीर्तितौ॥ चान्द्रांस्त्यजन्ति कालेन ह्यृतवः क्रान्तिसंश्रयाः। विषुवद्यत्र राशौ स्याद्वसन्तस्तत्र वै भवेत्॥ तत्समीपश्च यश्चान्द्रस्सो मधुर्यौगतस्तदा। चैत्रादिषु मधुः कस्स्यादित्यतो भाति सायनात्॥ ```

### ***॥धर्मशास्त्रप्रयोजनम्॥***

``` wp-block-verse अयनर्तुप्रयुक्तानि त्वनुष्ठानानि यानि वै। भास्करायनयोश्चैव क्रान्तावाधारितत्वतः॥ अयनाधारितत्वाच्चैवर्तूनां तेषु कर्मसु। क्रान्तेर्निर्णयनीयत्वात्सायनगणितेन च॥ शरदि वाजपेयेन यजेतेत्यादिषूक्तिषु। ऋतुसम्बद्धकत्वाच्च प्रयोजकस्य तत्त्वतः॥ गणनं सायनं कुर्यात्तिथिनिर्णयहेतुकैः॥ ```

### ***॥अयनांशपरिवर्तनम्॥***

``` wp-block-verse दृश्येते नानुनक्षत्रौ प्रत्यक्षेनायनौ यतः। तत्सङ्क्रान्ती तु नीतव्ये सायनान्नात्र संशयः॥ अयनांशश्च कालेन वर्द्धमानो हि दृश्यते। शून्यतोऽष्टदशस्तस्मादेकविंशतिरेव च॥ प्रोक्ता एवं यतः पूर्वैर्गणकैस्स्वीयवीक्षणात्। स चाद्य वेदनेत्रांश्च वर्धितः कालयापनात्॥ सङ्क्रान्तेस्सायनस्याथ भ्रमणं कीदृशं भवेत्। विद्वत्स्वत्र मतिद्वैधादूहे द्वे कीर्तिते बुधैः॥ आन्दोलनात्मकं कैश्चित्कैश्चिच्च भ्रमणात्मकम्। आन्दोलनात्मकञ्चेत्तद्गणितेन न सिद्ध्यति॥ सरसालाबुबद्धस्य सूत्रेणान्दोलनस्य तु। उपमा गणकैः कृत्वा तदसाध्यं प्रदर्शितम्॥

स्वमतञ्च वदिष्यामि ह्यर्वाचीनैर्यथोदितम्। जायते ह्ययनांशस्तु पृथिव्या अक्षवर्तनात्॥ चक्रवत्सततं तस्य रोधनाय न किञ्चन। यथाचक्रञ्चलत्येव कुतो मेषे पुनर्विशेत्॥ ```

## ***॥गणितानामैक्यावश्यकत्वम्॥***

``` wp-block-verse परेऽहन्यभवद्वादो भिन्नत्वे वाक्यदृश्ययोः। अद्वैतेऽर्कहिमांशूनां द्वैधमेवं कुतो भवेत्॥ गणनाबहुले चापि फलमेकं प्रसाधयेत्। यथा पथेष्वनेकेषु मुम्बायै निर्मितेष्वपि॥ नयन्त्येकत्र सर्वेऽपि कश्चित्कर्णावतीं तु न। अतो दृग्वाक्यकर्तारावैक्यमेतुं प्रचोदितौ॥ ```

### ***॥प्राचीनगणितानां शोधनावश्यकत्वम्॥***

``` wp-block-verse रामाचार्यस्तदा प्राज्ञो युक्तियुक्ता वचोऽब्रवीत्। अयनांशस्तु सर्वैर्वै पूर्वाचार्यैस्सदा युतः॥ मात्रं निरयणेनैव पञ्चाङ्गं नैव शक्यते। लग्नस्यानयनं यस्मान्नो शक्यं सायनं विना॥ तथा निरयणञ्चैव न त्यक्तुं सम्भवेद्बुधः। भगणा राशिमासाश्च युज्यन्ते तावदेव हि॥ उभावतः प्रयुज्येतां यथा साक्षात्प्रवीक्ष्यते। अधुनेच्छंस्तु मुम्बायै यातुं तैर्वै‌ पथैरियात्॥ प्राप्नोति न हि मुम्बायाञ्चरन्कर्णावतीपथे। वाक्यानि शोधितव्यानि प्रत्यक्षान्विति निशिचयः॥ अन्ये ये चैव युज्यन्ते प्राचीनगणिता इह। समेषामनिवार्येण साध्या दृक्तुल्यता बुधैः॥ नातस्त्याज्याः पुराणास्तु रीतयो ग्रहसाधने। तासां संशोधनं कृत्वा ताभिर्दृष्टान्प्रसाधयेत्॥ एतावच्छोधनञ्चैव नालमद्यकृतं भवेत्। भूयो भूयो हि कर्तव्यं युगानां परिवर्तनात्॥ प्रोक्तमेवं पुराचार्यैर्यैः प्रोक्ता रीतयस्त्विमाः। सिद्धान्तानां प्रमाणत्वं क्वचित्काले हि कीर्तितम्॥ कालान्तरे तु संस्कारश्चिन्त्यतां गणकोत्तमैः। यथोक्तं नीलकण्ठेन भेदे गणितदृश्ययोः॥ यन्त्रैर्दृग्भगणास्साद्ध्वार्वाचीनं गणितं करेत्। एवं संस्कारयित्वैव भवेम लोकयोर्यतः॥ ऐहिकेऽहसनीयाश्चादण्डनीयाः परे वयम्॥

अब्रवीदीश्वराचार्यो रोचमाना वचस्तदा। शोधयित्वेह यत्सिद्धं गणितं केरलेषु तु॥ विकारः पद्धतौ नात्रापेक्षितो दृश्यसाधने। सङ्ख्यानां भगणादीनां केवलं स्याद्विवर्तनम्॥ प्रतिषष्ट्यब्दिके कालेऽनिवार्यं प्रभवादिके। भगणाद्यास्तु संशोध्या यथा दृक्तुल्यता भवेत्॥ उपायद्वयमस्यापि सम्भवेच्छृणुतेह तौ। पाश्चात्यैर्ये प्रदीयन्ते भोगाद्या ज्यौतिषामिह॥ तत्स्वीकृत्यैव कुर्याम भगणादिप्रसाधनम्। अथवा पूर्वसिद्धान्तेष्वाचार्यैश्चोदितं यथा॥ वेधशाला यथाशास्त्रं निर्मायाभारते पुनः। द्वादशाङ्गुलिकं शङ्कुं स्थापयित्वा च तत्र नः॥ ग्रहानीक्षामहै तस्मात्सात्स्यामो भगणान्स्फुटान्॥
