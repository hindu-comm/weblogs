How the original Iron Man should have ended

If you haven’t checked out the “How It Should Have Ended” [website](http://www.howitshouldhaveended.com/), I really recommend it. I’m not a big consumer of web video excepting *South Park* and *The Onion*, but this is really good stuff. In particular this [alternative ending of the original](http://www.howitshouldhaveended.com?bcpid=42616701001&bclid=38536157001&bctid=80183803001) *Iron Man* is pretty amusing.
