Manjoo on Ubuntu

[Linux Is Making Me Insane: Grappling with Ubuntu, the free, open-source operating system](http://www.slate.com/id/2202664/). I have Ubuntu on my PC through a dual-boot. I also purchased a USB wireless card which was guaranteed to be compatible with Ubuntu plug-and-play. It does work…80% of the time. The problem is that it “drops” the connection every half hour or so and I might have to end up rebooting the system to get it to work again. No thanks. There are some programs which only run on Linux systems that I keep Ubuntu around for (VMware is way too slow), but for browsing the internet it is just not useful for me. If someone like me, or [Thomas Mailund](http://www.mailund.dk/index.php/2008/10/16/how-did-i-ever-consider-linux-stable/), is lukewarm to the most user-friendly of the Linux distributions, I’d say not ready for primetime as your grandma’s OS….
